Split Image 是一个专门为处理 **Nano Banana** 生成的表情包而设计的图片处理工具，提供了一站式的切片和去底解决方案

## Features

- **智能切片**：可视化操作，将长图切分为单个表情，支持自定义分割位置
- **快速去底**：去除白色背景

## Usage

1. **切片**：上传 Nano Banana 生成的长图，添加分割线并导出
2. **去底**：将切好的图片拖入"去白底"页面，使用橡皮擦涂抹去除白色背景
